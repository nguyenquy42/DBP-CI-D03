Patterned Wristband

Viết một hàm trả về true nếu phần có thể được phân loại chính xác thành một trong 4 loại và false nếu không.



Ví Dụ:
isWristband([ ["A", "A"], ["B", "B"], ["C", "C"] ]) ➞ true // Part of horizontal wristband.
isWristband([ ["A", "B"], ["A", "B"], ["A", "B"] ]) ➞ true // Part of vertical wristband.
isWristband([ ["A", "B", "C"], ["C", "A", "B"], ["B", "C", "A"], ["A", "B", "C"] ]) ➞ true // Part of diagonal left wristband.
isWristband([ ["A", "B", "C"], ["B", "C", "A"], ["C", "A", "B"], ["A", "B", "A"] ]) ➞ true // Part of diagonal right wristband.
